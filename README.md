# 🎓 Student Dropout Predictor (AI-Based)

An AI-powered web application that predicts whether a student is at risk of dropping out based on academic and financial data.

---

## 📌 Features

- 👤 Student Information Input (Name, Registration No, Department, Course)
- 💰 Financial Status Check (Fees & Scholarship)
- 📊 Academic Performance Analysis (Semester-wise)
- 🤖 AI Prediction using Machine Learning
- 📈 Dynamic Semester Addition
- 🎯 Smart Risk Detection (High Risk / Continue / Graduate)
- 🎨 Modern UI with Gradient & Glass Effects

---

## 🧠 How It Works

The system uses a trained Machine Learning model to analyze:

- Academic performance (subjects passed, grades)
- Financial status (fees payment)
- Scholarship availability

Based on this data, it predicts:

- ⚠️ High Dropout Risk  
- 📚 Likely to Continue  
- 🎓 Likely to Graduate  

---

## 🛠️ Tech Stack

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Python (Flask)  
- Machine Learning Model (Scikit-learn)
